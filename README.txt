This is a program that can parse xlsx files.

How to use it?

-Download Node.js on your computer from this link to be able to run this program on your computer: https://nodejs.org/en/download/
-Check if Node.js has been installed by typing "node -v" in cmd
-Run a program according to your OS (Parse.exe for Windows, ParseLinux for Linux, ParseMacOS for MacOS)
-Follow the instructions you receive in cmd
-When the work is done, files with parsed information will be in the "parsedFiles" folder
